# 🧳 Job Portal

A dynamic job portal web application developed using PHP and CodeIgniter. It connects job seekers with employers and allows easy job postings, applications, and profile management.

---

## 🚀 Features

- 👤 User Registration & Login (Job Seeker and Employer)
- 📝 Resume upload and profile builder
- 🔍 Job search with filters (location, category, salary, etc.)
- 📢 Employers can post and manage jobs
- 📨 Job seekers can apply for jobs directly
- 📊 Admin Dashboard to manage users and listings
- 📅 Recent Jobs & Featured Job Listings on homepage

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap
- **Backend:** PHP (CodeIgniter Framework)
- **Database:** MySQL
- **Server:** XAMPP (Apache)
